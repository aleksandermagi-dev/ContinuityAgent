## Full Product Description

```text
AI tools can write code fast, but they still forget why your project is shaped the way it is.

Continuity Layer is a local-first desktop app for AI builders who want shared project memory for humans and agents. It scans a project folder, detects useful docs and validation checks, tracks decisions and unresolved branches, generates health/drift reports, and exposes compact continuity packets through CLI and MCP.

Use it when you want an assistant to understand:
- what the project currently is
- what changed recently
- why decisions were made
- what checks should be run
- what branches are unresolved
- what risks or stale assumptions need review

Agent writes are review-safe. Updates create drafts until you accept them. Detected commands are recommended, not auto-run. Project files are not silently mutated.

Early access includes the Windows desktop build, local API/CLI/MCP access, setup docs, demo workflow, and a feedback path while the product is still being shaped with real users.
```
